Data Quality Report (DQR)¶

Vítor Moura - Data Scientist¶

  • DQR um relatrio analtico com o objetivo de compreender a organizao dos dados, se esto coerentes e se esto consistentes.
  • Possibilitando identificar problemas e corrigi-los, encontrar anomalias e resumir os dados com base no problema de negcio.

# Funo para categorizao da coluna
def imc(c):
    if c <= 18.5:
        return 'Abaixo do Peso'
    elif c >= 18.6 and c <= 25:
        return 'Peso Ideal'
    elif c >= 25.1 and c <= 30:
        return 'Sobrepeso'
    elif c >= 30.1 and c <= 35:
        return 'Obesidade Grau I'
    elif c >= 35.1 and c <= 39.9:
        return 'Obesidade Grau II'
    elif c >= 40:
        return 'Obesidade Grau III'
